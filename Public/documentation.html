myFlix

Welcome to myFlix, a movie database where you can search for movies, access information about movies, directors, and genres. You can also create a profile and add them to your favorites list.

myFlix API Documentation

Request URL HTTP Method Request Body Response Body
Get list of all movies /movies GET A JSON object holding an array with all movies
Get data about a single movie by title /movies/:title GET none A JSON object holding data about a single movie, including the description, director, genre, image URL, and whether its featured property or not. Example: { "Title": "Jurassic Park", "Description": "A pragmatic Paleontologist visiting an almost complete theme park is tasked with... Get data about a genre by name /movies/genres/:genreName GET none A JSON object holding data about a genre, including name and description. Example: { "Name": "Action", "Description": "Action films usually include high energy, big-budget physical stunts and chases..." }
Get data about a director by name /movies/directors/:directorName GET none A JSON object holding data about a director, including name, bio, birth year, and death year.
