myFlix Documentation

This application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request Body Response Body
Return a list of ALL movies to the user /movies GET / Read N/A N/A
Return data (description, genres, year of release, image URL) about a single movie by title to the user /movies/:title GET / Read [title] N/A
Return data about a genre (description) by the name or title ("Halloween") /movies/genres/:genreName GET / Read [genreName] N/A
Returns data about a director (short bio) by name /movies/directors/:directorName GET / Read [directorName] N/A
Allows new users to register /users Post / Create N/A JSON object with username, password, email, birthday
Allows users to update their user info (username) /users/:username PUT / Update [username] JSON object with username, password, email, birthday
Allows users to add a movie to their list of favorites /users/:username/movies/:movieTitle POST / Create [username, movieTitle] N/A
Allows users to remove a movie from their list of favorites /users/:username/movies/:movieTitle DELETE / Delete [username, movieTitle] N/A
Allows existing users to deregister /users/:username DELETE / Delete [username] N/A
