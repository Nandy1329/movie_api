Welcome to myFlix, a movie database where you can search for movies, access information about movies, directors, and genres. You can also create a profile and add them to your favorites list.

Business Logic URL HTTP Method Query Params Request Body Response Body
Return list of all movies /movies GET none none return JSON object with all movie data
Return data about single movie by title /movies/:title GET :title none return JSON object with specific movie data
Return data about a genre type by name/title /movies/genre_description/:Genre GET :Genre none return JSON object with specific movie genre type
Return data about a director by name /movies/director_description/:Director GET :Director none return JSON object with details about a director
Allow new users to register /users POST none New user input info (username,email,password, birthday) in JSON format New user info that was added in JSON format
Return all users /users GET none none Return all registered user info
Allow users to update their username /users/:Username PUT :Username Enter updated info in JSON format (username, email, password, birthday) Updated user info (username) that was added in JSON format
Allow users to add a movie to their favorites /users/:Username/movies/:MovieID POST :Username :MovieID none Confirmation that the movie title has been added to favorites
Allow users to delete a movie from their favorites /users/:Username/movies/:MovieID Delete :Username :MovieID none Confirmation that the movie title has been deleted favorites
Allow users to deregister /users/:Username Delete :Username none Confirmation that user email has been removed
